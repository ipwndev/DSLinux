DSlinux is a project that aims to port Linux to the Nintendo DS gaming console.
The page you are reading now is hosted on a Nintendo DS, via DSlinux using the ported webserver.
DSlinux has come a long way as you can see, read more about it in the "about" section.

This website is shown so small to make sure small screens will also be able to render it correctly, please understand ;).
