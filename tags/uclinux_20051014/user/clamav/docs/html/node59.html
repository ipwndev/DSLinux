Gadoyanvirus

Homepage: http://oss.mdamt.net/gadoyanvirus/
Supports: libclamav
gadoyanvirus is a (yet another) virus stopper for qmail. It replaces the original qmail-queue program. It scans incoming messages using the ClamAV anti-virus library. Suspect message will be quarantined and (optionally) a notification message will be sent to the recipients. By default, gadoyanvirus needs QMAILQUEUE patched qmail installation.
