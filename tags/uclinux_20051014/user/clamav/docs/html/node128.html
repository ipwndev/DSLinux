QtClamAVclient

Homepage: http://www.xystumnet.com/qtclamavclient.html
Supports: clamd
A small client for ClamAV that uses the STREAM socket connection to a clamd server machine where the daemon is listening to locally scan files. It is based on the Qt Toolkit from Trolltech.
