Century Software Embedded Technologies provides custom solutions and technologies for embedded Linux that will enable you to quickly and efficiently meet the requirements of your vendors and application providers. Starting with a solid graphical windowing environment targeted at embedded systems, Century then builds complete, custom embedded applications for Internet appliances, PDAs, and turnkey systems requiring graphical displays or multimedia audio and video technologies.

Our technologies are targeted at small footprint applications and allow applications to be designed, prototyped, emulated and tested for the target device on the Linux desktop, which will enable the software design teams to move forward while the hardware design/manufacturing process proceeds.
