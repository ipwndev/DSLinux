pop3.proxy

Homepage: http://quietsche-entchen.de/cgi-bin/wiki.cgi/proxies/Pop3Proxy
Supports: clamd
pop3.proxy is a proxy server for the POP3 protocol. Usually it's used on a firewall between a client and a POP3 server taking care that both sides talk POP3 protocol as described in RFC 1939. There are some additional features beside normal proxying. pop3.proxy can grant or deny access based on an external access control program which receives some POP3 session information. pop3.proxy can also feed e-mails into a local ClamAV daemon checking the e-mails for viruses before passing them to the client.
